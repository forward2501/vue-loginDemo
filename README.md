# 一款基于Vue3.0模拟登录注册等的单页面应用

## 概述

本项目是使用Vue.js3.0框架，结合sass语法搭建的，主要提供登录和验证页面的切换以及逻辑验证，关注组件之间的数据传递等。

项目地址：https://github.com/forward2501/vue-loginProject

## 本地预览

1. 克隆代码到本地

`git clone git@github.com:forward2501/vue-loginProject.git`

2. 进入项目路径

`cd vue-loginProject`

3. 安装依赖

`npm install`

4. 启动项目

`npm run serve`
